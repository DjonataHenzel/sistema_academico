Projeto sistema academico
Configurações e ferramentas utilizadas para rodar projeto:
-> JDK 11
-> Spring Tool Suite 4
-> PosgreSQL 13
-> Maven 3.1.1
-> Configurações como nomeBanco e password deverão ser configurados na classe application.properties antes de executar o programa

OBS: Para rodar o projeto deverá ser executado a classe (SistemaAcademicoApplication). Nessa classe contem os Read, Delete e Update. Já os Insert's foram deixado na classe (InseridosDadosIniciais).
